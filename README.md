# CheckList App (Flutter)

A powerful and elegant shopping list application built with Flutter.

## Features

- **Multiple Shopping Lists**: Create and manage different lists for different needs.
- **Dynamic Renaming**: Rename your lists anytime directly from the list view.
- **Smart Categorization**: Organize items into categories with automatic reordering.
- **Persistent Storage**: Your lists and items are saved locally on your device.

## Usage

### Managing Lists
- **Create**: Tap the "Add" button in the center of the bottom navigation bar on the Home screen to create a new list.
- **Rename**: While viewing a shopping list, tap the **pencil icon** next to the list name in the top bar. Enter the new name and tap "Save". The name will update everywhere in the app.
- **Switch**: Select any list from the Home screen to view and manage its items.

### Managing Items
- **Add**: Tap "Nova Categoria" to add categories, or tap the "+" icon within a category to add items.
- **Check/Uncheck**: Tap or swipe items to mark them as completed. Completed items move to the bottom.
- **Move/Copy**: Long press or use item menus to reorganize your items across categories.
